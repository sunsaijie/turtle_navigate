# 寻路测试

以可视化的方式进行寻路算法的测试, 使用了python的 turtle标准库实现


## 使用方法

### `matrix.py`的文件

其他的loadmap变量为地图, 可以进行更改，数字代表的含义



| 数字| 含义| 颜色|
| --- | --- | --- |
| 0   | 空地| 白色|
| 2   | 河流| 蓝色|
| 6   | 地雷| 红色|
| 7   | 金矿| 黄色|
| 8   | 高山| 灰色|


points代表点的初始位置,为一个字段


```
{
   id_: ( x, y )
   ...
   ...
   ...
}

id_为数字，代表不同的点

x, y 代表在地图中的起始位置
```

### `run.py`的文件

填充Resolve类中的resolve方法,返回值

```
[
    {id1: [1,1], id2:[1,-1], id3: [0,1] ....}
    ...
]

1. 列表中的每个字典，代表每一轮的信息
2. 每个字典，键为点的id值，代表不同的点，值为一个方向向量，只有8个方向
	分别为(1,1), (-1,-1), (1,-1) , (1,0) ...

```
